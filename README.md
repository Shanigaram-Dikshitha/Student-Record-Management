# Student Record Management (Java Swing)

A simple and user-friendly **Java Swing desktop application** designed to manage student records efficiently.  
The system allows users to add, update, delete, and view student details using a graphical interface.  
All data is stored in a CSV file, ensuring that student records remain available even after restarting the application.
